# 项目介绍

项目是MasRobo RCS SDK，包括Python、NodeJS、Java等语言的SDK。

GO语言SDK在另外一个项目中，项目地址为：https://github.com/masrobo/rcs-golang-sdk

# 有调整时的处理

需要在各个语言的SDK中进行调整。

# 发布规则

1. 修改版本号，首先修改VERSION文件，接着在各个语言的SDK中修改版本号
2. 发布前，确保本地代码和线上代码一致
3. golang项目需要推送到远端github仓库
4. 最后发布各个语言的SDK
- 发布Python SDK
- 发布Node.js SDK
- 发布Java SDK
- 发布GO语言SDK（rcs-golang-sdk）
