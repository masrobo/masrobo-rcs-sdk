# app_id & app_key

To obtain the `app_id` and `app_key`, please follow these steps:
1. Visit the [MASROBO Developer Center](https://developer.boticz.cn/)
2. Register for an account if you don't have one, or log in if you already have an account.
3. Navigate to the [Applications](https://developer.boticz.cn/application/list) section.
4. Create a new application by filling in the required information.
5. Once the application is created, you will find the `app_id` and `app_key` in the application details.
